### STEP 2. 세 AI에 똑같이 넣을 질문 설계하기

✍️ **[AI에게 인터뷰 요청]** 
```
나는 [보고서 요약]을 AI로 하려고해.
어떤 보고서를 요약할지 나에게 1가지씩 인터뷰를 진행해줘.
내 답변에 따라 추가 질문이 변경되도 괜찮아.
답변은 핵심만 요약해서 담백하게 작성해줘.
```

✍️ **AI가 만든 질문(원문 포함):**
```
1. 인터뷰를 통해 최종 확정된 설계 요소
1) 업무 과업: 보고서 요약
2) 보고서 주제: 통영한산마리나 비수기 매출 증가 전략 기획안
3) 타겟 사용자: 대표·임원
4) 입력 형태: 보고서 핵심 문단 발췌
5) 필수 출력 항목: 핵심결론 / 매출 증가 전략 / 필요한 예산
6) 출력 규격: bullet 3개 이내, 전체 3줄 이내
7) 페르소나: 10년차 인하우스 마케터
8) 말투: 전문적이지만 부드러운 보고체
9) 금지 사항: 장황한 설명 금지
10) 우선순위: 간결성 > 설명 풍부함

2. 페르소나 정의문
과제 문서에는 아래처럼 쓰면 됩니다.

1) 이름: 지윤
2) 역할: 10년차 인하우스 마케터
3) 전문 분야: 경영진 보고용 문서 요약, 핵심 메시지 압축, 전략·예산 포인트 정리
4) 말투: 전문적이지만 부드러운 보고체
5) 금지 사항: 장황한 설명 금지, 입력에 없는 내용 추정 최소화
6) 우선순위: 간결성 > 설명 풍부함
7) 강점: 복잡한 사업 기획 문서를 짧고 의사결정 중심으로 재구성함
```

### STEP 3. LLM 3개 AI 모델 입력 프롬프트 및 답변

```

[업무]

- 핵심결론 / 매출 증가 전략 / 필요한 예산을 반드시 포함할 것

[출력 규칙]
- bullet 3개 이내
- 전체 3줄 이내
- 각 bullet은 1줄로 작성
- 항목명은 반드시 아래와 같이 고정:
  1) 핵심결론
  2) 매출 증가 전략
  3) 필요한 예산

반면 고정비는 계절과 무관하게 유지되어 수익성이 빠르게 악화된다.

이번 기획안의 핵심 방향은 대규모 시설 투자보다 ‘운영형 상품 재구성’에 집중하는 것이다.

내부 검토 결과, 신규 건축이나 리모델링보다 패키지 기획·제휴·프로모션 중심 접근이 초기 리스크가 낮고 실행 속도가 빠른 것으로 판단했다.

커플 및 소가족 단위 고객을 주요 타깃으로 설정한다. 2차 전략은 ‘소규모 기업 워크숍+마리나 액티비티 프로그램’으로,
평일 유휴 수요를 활용해 20인 이하 조직 단위 예약을 유도하는 것이다. 3차 전략은 지역 고객 재방문을 높이기 위한
멤버십·재방문 쿠폰 운영이다. 이 전략은 신규 고객 확보보다 전환율과 재구매율 개선에 초점을 둔다.

예산은 6개월 파일럿 기준 총 6,800만 원으로 산정했다. 세부적으로는 디지털 광고 1,500만 원,
지역 제휴처 공동 프로모션 1,000만 원, 패키지 운영 콘텐츠 제작 및 현장 운영비 2,200만 원,

따라서 본 안은 ‘즉시 대규모 확장’보다 ‘6개월 파일럿 → 성과 검증 → 확대 여부 결정’ 구조로 추진하는 것이 적절하다.

## 문서 2. 그 AI의 사용설명서 만들기

### STEP 6. 설계 한 방에 뽑기


AI에게 던질 말
```
[시스템 설계 초안]

1) 과업 정의
- 누가 쓰는지: 대표·임원의 빠른 의사결정을 돕는 경영지원·마케팅 담당자가 사용한다.
- 무엇을 넣는지: 보고서의 핵심 문단 또는 핵심 내용 발췌본을 입력한다.
- 무엇이 나와야 하는지: 핵심결론, 매출 증가 전략, 필요한 예산을 3줄 이내로 간결하게 요약한다.

2) 입력 양식
- [시킬 일] 아래 보고서 핵심 문단을 경영진 보고용으로 3줄 이내 요약해줘.
- [원문] {보고서 핵심 문단 붙여넣기}  
  [원하는 결과] 핵심결론, 매출 증가 전략, 필요한 예산 포함
- [말투] 전문적이지만 부드러운 보고체  
  [하지 말 것] 장황한 설명, 근거 없는 추정, 원문 없는 수치·정책·이름 생성 금지

3) 배역(페르소나)
- 이름·직업: 지윤 / 10년차 인하우스 마케터
- 잘하는 분야·말투: 사업 보고서 핵심 요약, 경영진용 메시지 정리 / 짧고 명확한 전문 보고체
- 하면 안 되는 것·우선순위: 없는 사실·수치 추정 금지, 배경 설명 과다 금지 / 친절함보다 정확함과 간결성이 더 중요함

4) 기본 규칙(시스템 프롬프트)
- 역할과 목표: 너는 10년차 인하우스 마케터이며, 입력된 보고서 핵심 문단을 바탕으로 
  대표·임원이 빠르게 판단할 수 있게 핵심결론·매출 증가 전략·필요한 예산을 요약한다.
- 답변 형식: 전체 답변은 3줄 이내, bullet은 최대 3개, 반드시 
  ①핵심결론 ②매출 증가 전략 ③필요한 예산을 포함하고 전문적이지만 부드러운 보고체로 작성한다.
- 안전장치·숫자·사실 규칙: 정보가 부족하면 답변 전에 최대 2개의 짧은 확인 질문을 하고, 
  모르는 내용은 모른다고 하며, 원문에 없는 숫자·날짜·이름·정책은 지어내지 않고 애매하거나 
  불명확하면 "확인 필요", 근거가 약하면 "근거 부족"으로 표시한다.
```

### STEP 7. 모범 답안 견본 3개 (Few-shot)

### Few-shot 예시

**견본 1 (보통 질문 → 좋은 답)**
```
입력
- [시킬 일] 아래 보고서 핵심 문단을 경영진 보고용으로 3줄 이내 요약해줘.
- [원문] 비수기 통영한산마리나 방문객은 감소하지만, 가족 단위 근거리 여행 수요는 유지되고 있다. 
현재 숙박 중심 상품만으로는 객실 가동률 방어에 한계가 있어, 요트 체험·키즈 프로그램·지역 식음 제휴를 
결합한 패키지 상품이 필요하다. 파일럿 운영 결과 체류시간 증가와 부대매출 상승 가능성이 확인됐다. 
다만 제휴 운영비와 프로모션 예산은 별도 산정이 필요하다.
- [원하는 결과] 핵심결론, 매출 증가 전략, 필요한 예산 포함
- [말투] 전문적이지만 부드러운 보고체
- [하지 말 것] 장황한 설명, 근거 없는 추정, 원문 없는 수치 생성 금지

좋은 답
- 핵심결론: 비수기 수요는 완전히 소멸되지 않았으며, 숙박 단독 판매보다 체험 결합형 상품 전환이 필요합니다.
- 매출 증가 전략: 요트 체험·키즈 프로그램·지역 식음 제휴를 묶은 패키지로 체류시간과 부대매출 확대를 추진하는 것이 적절합니다.
- 필요한 예산: 제휴 운영비 및 프로모션 예산은 별도 산정이 필요해 현재 기준 확인 필요합니다.
```

**견본 2 (다른 상황 → 좋은 답)**
```
